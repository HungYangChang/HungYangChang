# Repository Scorecard

Use this rubric to evaluate candidate repositories for pinning.

## Scoring Rubric (1–5 per criterion)

| Criterion | Weight | Description |
|-----------|--------|-------------|
| **Relevance** | 30% | How directly does this repo demonstrate target-role skills (AI/LLM, backend, production)? |
| **Completeness** | 25% | Does it have a full README, setup instructions, architecture docs, and working demo? |
| **Quality Signal** | 25% | Are there tests, CI, clean code, proper git history, and meaningful commits? |
| **Demo Readiness** | 20% | Can a recruiter see the project in action (live link, screenshots, GIF, video)? |
